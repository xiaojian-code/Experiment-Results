This project is an experimental presentation about the paper named Distributed Finite-time Cooperative Pursuit of Multi-UAV Systems Based on Local Information. The experiment video is in the Experiment_result folder. 
In addition, Experiment_result folder also contains the experimental platform and the experimental framework.
There is a txt file in Experiment_result which mainly notes on some of the acronyms in the experimental video.
Notes.1：
Scheme 1: Finite-time control scheme with finite-time observer（FTO）;
Scheme 2: Finite-time control scheme with Kalman filter（KF-LI）;
Scheme 3: Finite-time control scheme with linear observer（LO）.
