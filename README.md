This project is an experimental presentation about the paper named Distributed Finite-time Cooperative Pursuit of Multi-UAV Systems Based on Local Information. The project includes Experiment video,  Experimental platform and Experimental framework.
In addition, there is a txt file which mainly notes on some of the acronyms in the experimental video.
Notes.1：
Scheme 1: Finite-time control scheme with finite-time observer（FTO）;
Scheme 2: Finite-time control scheme with Kalman filter（KF-LI）;
Scheme 3: Finite-time control scheme with linear observer（LO）.
